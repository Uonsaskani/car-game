# جاده — بازی رانندگی با ماشین‌های ایرانی
سازنده: یونس

## چی توش هست
- منوی گاراژ: تعمیر ماشین / سفارش قطعات / خرید ماشین / شروع بازی
- ۴ ماشین: پژو ۴۰۵ جی‌ال‌اکس، پژو پارس، زانتیا، تویوتا سری ۷۰
- تقویت قطعات (موتور، توربو، اگزوز، ECU، لاستیک، کمک‌فنر، جعبه‌دنده) تا بالای ۲۶۰ کیلومتر بر ساعت
- رانندگی اول‌شخص با فرمان و دست‌های متحرک، دنده‌ی دستی
- یک مسیر بسیار طولانی با پیچ‌های ۳۰/۹۰/۱۸۰ درجه، بخش‌های تک‌لاین و دولاین، جاده‌ی خراب، سرعت‌گیر
- چرخه‌ی خودکار روز و شب
- صدای موتور و جیغ لاستیک سرِ پیچ‌های تند، وابسته به سرعت واقعی

## تست سریع روی خودِ لپ‌تاپ (بدون هیچ نصبی)
فقط پوشه‌ی `www` رو با افزونه‌ی Live Server توی VS Code باز کن (راست‌کلیک روی
`www/index.html` → Open with Live Server) یا با هر مرورگری فایل `www/index.html`
رو باز کن. بازی کامل توی مرورگر اجرا می‌شه (اینترنت لازمه چون فایل Three.js
از CDN لود می‌شه).

کنترل تست با کیبورد: جهت‌نماها یا W/A/S/D، دنده با دکمه‌های روی صفحه.

## گرفتن APK بدون نصب Android Studio
این پروژه با **Capacitor** ساخته شده و ساخت APK به‌طور کامل روی سرورهای
**GitHub Actions** (ابری، رایگان) انجام می‌شه — لپ‌تاپ تو فقط کد رو push
می‌کنه و چیزی دانلود/نصب نمی‌کنه.

مراحل:
1. یه ریپازیتوری روی GitHub بساز و کل این پوشه (`car-game`) رو داخلش push کن.
2. برو به تب **Actions** توی همون ریپو → روی workflow به اسم
   **Build Android APK** بزن → **Run workflow**.
3. چند دقیقه صبر کن تا build سبز بشه، بعد از پایینِ همون صفحه، از بخش
   **Artifacts** فایل `jadde-debug-apk` رو دانلود کن — یه فایل zip هست که
   `app-debug.apk` توشه.
4. اون APK رو روی گوشیت منتقل کن و نصبش کن (باید نصب از منابع ناشناس رو
   توی گوشی فعال کنی).

نکته: بار اول که workflow اجرا می‌شه، دستور `npx cap add android` پوشه‌ی
`android/` رو خودش می‌سازه؛ لازم نیست تو چیزی دستی بسازی.

## بعدش چی؟
این نسخه یه اسکلت کامل و قابل‌بازی‌ست، نه نسخه‌ی نهایی. قدم بعدی طبیعی اینه که:
- مدل‌های سه‌بعدی واقعی‌تر برای ۴ ماشین (به‌جای جعبه‌های رنگی فعلی) اضافه بشه
- فایل صوتی واقعی موتور/اگزوز (که خودت ضبط یا تهیه کنی) جایگزین صدای سنتزی بشه
- تعادل اقتصادی قیمت‌ها (سکه/قیمت قطعات) بر اساس تست واقعی روی گوشی تنظیم بشه

هر کدوم از این‌ها رو بخوای جلو ببریم بگو تا ادامه بدیم.
